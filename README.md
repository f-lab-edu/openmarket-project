# openmarket-project
다수의 판매자와 소비자가 온라인 상에서 상거래를 할 수 있는 온라인 거래 플랫폼

# 🛠️ 기술 스택
- Java 17
- Spring Boot 3.4.4
- Gradle
- JPA / Spring Data JPA
- MySQL 8.0
- Spring Batch 5.2.2
- Redis 8.0.1
